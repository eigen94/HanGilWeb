예제 페이지 입니다.

    코스타111기 웹 프로젝트는 웹표준을 지키지 않은 홈페이지를 웹표준을 지킨 페이지로 리뉴얼 하는 것이고, 리뉴얼 대상으로 한길사 페이지를 선정했습니다.

     지금 보고 있는 이 페이지는 부트스트렙에 있는 탬플릿을 사용했습니다. 이 레이아웃을 기반으로 한길사에 어울리는 디자인으로 변경하는 작업을 하려합니다. 현 상태에서 변경하길 원하는 부분과 새로이 적용하고 싶은 부분과 관련된 의견은 언제든지 연락주시길 바랍니다.
     작업을 하실때는 web폴더에 새로운 html파일을 만들어서 index.html파일 내용을 전부 복사한 후 붙여 넣고 컨테이너 안에 표시한 영역 안에서 마음껏 페이지를 제작하시면 됩니다. jQuery파일은 javascriptFiles폴더에 위치하고 있으며 각자 생성한 javascript파일은 이 javascriptFiles폴더에 저장하면 됩니다. 마찬가지로 새로이 만든 css파일은 cssFiles폴더에 저장하면 됩니다.
     부트스트랩을 이용하여 반응성 웹이 되도록 클레스에 부트스트랩 기능을 지정해 주세요. 자신만의 스타일을 적용하고 싶을 때는 부트스트렙폴더안에 있는 css폴더에 새로운 css파일을 만들고 20번째 줄에 bootstrap/css/index-theme.css 대신 새로만든 css파일명을 적어서 스타일을 적용하면 됩니다.
     스타일을 적용 할 때는 컨테이너 내부의 스타일만 적용해주시길 바랍니다. 컨테이너 외부의 디자인은 다같이 상의 후에 적용하기로 하겠습니다. 다들 마지막 프로젝트 즐겁게 진행합시다! 화이팅!
